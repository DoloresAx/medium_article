Ground Plane Designing ချရင် ဘာတွေ သိထားဖို့ လိုမလဲ? 🤔

Type of Ground Plane 🔄

Solid Ground Plane

-PCB Board မှာ Copper Plane ဟာ Gap မရှိဘဲ တစ်ဆက်တည်း ရှိနေတဲ့ အမျိုးအစားဖြစ်ပါတယ်။

-Signal Integrity နဲ့ Noise Reduction အတွက် အကောင်းဆုံးဖြစ်ပါတယ်။

Split Ground Plane

-လိုအပ်ချက်အရ Ground Plane တွေကို တစ်ခုနဲ့ တစ်ခု သီးခြားစီ ခွဲထားတဲ့ အမျိုးအစားဖြစ်ပါတယ်။

-အဓိကအားဖြင့် Mixed Signal Design (Analog + Digital) တွေမှာ အသုံးပြုပါတယ်။

Size 📏

-Ground Plane Area ကြီးလေ Impedance နဲ့ Resistance လျော့ကျလေ ဖြစ်ပါတယ်။

-EMF Noise တွေကို ပိုပြီး လျော့ကျစေပြီး Efficiency ပိုကောင်းစေပါတယ်။

-High Frequency Analog and Digital Design တွေမှာ Ground Plane ဟာ Board တစ်ခုလုံးကို Cover ဖြစ်အောင် ထားသင့်ပါတယ်။

Routing 🛤

-Track Line တွေဟာ Ground Plane နဲ့ တတ်နိုင်သမျှ နီးအောင် ထားသင့်ပါတယ်။

-ဒါမှ Signal Reflection နဲ့ Noise တွေ လျော့ကျမှာဖြစ်ပါတယ်။

Via Placement 🔗

-Multi-layer Ground Plane တွေမှာ Connection ကောင်းအောင် Via Hole တွေကို စနစ်တကျ ထားဖို့ လိုပါတယ်။

-Via Hole တွေ တစ်ခုနဲ့ တစ်ခု အရမ်းမဝေးဖို့ လိုပါတယ်။ အရမ်းဝေးရင် Return Path မှာ Weak Path တွေ ဖြစ်လာနိုင်ပါတယ်။

-Via Stitching လို နည်းလမ်းမျိုး (Via Hole တွေ အများကြီး ဖောက်ထားတာ) ကို အသုံးပြုသင့်ပါတယ်။

Avoid Large Holes 🚫

-Ground Plane ပေါ်မှာ Mounting Hole လို Size ကြီးတဲ့ Hole တွေ ဖောက်တာမျိုး ရှောင်သင့်ပါတယ်။

-High Impedance နဲ့ Noise တွေ ဖြစ်စေပြီး Signal Integrity ကို ထိခိုက်စေနိုင်ပါတယ်။

Multi-layer PCB Design 🎛

-Multi-layer PCB တွေမှာ Ground Plane နဲ့ Power Plane တွဲပြီး ထားသင့်ပါတယ်။

-ဒါမှ Return Path နဲ့ Power Distribution တည်ငြိမ်မှာ ဖြစ်ပါတယ်။

Ground Plane Split 🔄

-Analog နဲ့ Digital Circuit တွေ ပါတဲ့ PCB Design တွေမှာ Signal Interference မဖြစ်အောင် Ground Plane တွေကို သီးခြားစီ ခွဲထားသင့်ပါတယ်။

-ကျန်တဲ့ အခြေအနေတွေမှာတော့ Ground Plane ကို တစ်ဆက်တည်း (Gap မရှိ) ထားသင့်ပါတယ်။

Decoupling Capacitors ⚡️

-Capacitor တွေဟာ Voltage တည်ငြိမ်အောင် နဲ့ Noise တွေ လျော့ချဖို့ အတွက် အရေးကြီးပါတယ်။

-Components တွေရဲ့ Power Pin တွေနဲ့ အနီးဆုံး ထားသင့်ပြီး၊ Capacitor တွေရဲ့ Return Path ကို Ground Plane နဲ့ တိုက်ရိုက် ချိတ်ဆက်ထားသင့်ပါတယ်။
